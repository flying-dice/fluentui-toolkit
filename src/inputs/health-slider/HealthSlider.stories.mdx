# Health Slider Input

Health Slider Component, takes a Health Between 0 and 1 and provides a 0-100% slider.

Used for representing Health in Editable Menus

## API

The API Extends the default Slider API with Health and an onChange listener

See https://developer.microsoft.com/en-us/fluentui#/controls/web/slider

- `health` - Current Health, between 0 and 1
- `onChange` - Callback invoked when the slider changes (this is debounced to 250ms)
